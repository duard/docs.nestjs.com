Overview

Purpose

Here to help people to have a nice project scaffolding and Nest architecture component generator, the @nestjs/schematics library can help you more than expected.
The @nestjs/schematics library is based on the @angular-devkit projects.
The purpose is expose a collection of schematics to scaffold generic projects with a standard Nest structure.

@nestjs/schematics project follow the Nestjs versions.

Contribute

You can participate an contribute to the project on this GitHub repository : @nestjs/schematics
