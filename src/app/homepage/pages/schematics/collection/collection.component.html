Collection

As explain in overview section the @nestjs/schematics expose a collection of schematics described below.

application

The application schematic generates a standard Nest project structure.

Option Name Description Required Default Value
directory The directory name to create the app in. true N/A

module

The module schematic generates a Nest module.

Option Name Description Required Default Value
name The name of the module. true N/A
path The path to create the module. false src

controller

The controller schematic generates a Nest Controller.

Option Name Description Required Default Value
name The name of the controller. true N/A
path The path from to create the controller. false src

The service schematic generates a Nest Service.

Option Name Description Required Default Value
name The name of the service. true N/A
path The path from to create the service. false src
