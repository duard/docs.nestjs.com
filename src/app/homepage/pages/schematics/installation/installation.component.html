Installation

You need first to install the @angular-devkit/schematics-cli globally first : 

    
      $ npm install -g @angular-devkit/schematics-cli

Next you have to git clone and build the @nestjs/schematics :

Docker 
    
      $ git clone https://github.com/nestjs/schematics.git
      $ cd schematics
      $ make prepare
      $ make build
