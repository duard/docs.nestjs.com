Installation

ATTENTION! NestjsCLI follow the Nestjs major versions since v4. Be aware to have same NestjsCLI and Nestjs major versions installed.

From NPM registry

    
      $ npm install -g @nestjs/cli@[version]

From Docker Hub

Available soon.

Build yourself

Having Docker installed on your machine then git clone the @nestjs/cli GitHub repository : 

    
      $ git clone https://github.com/nestjs/cli.git
      $ cd cli
      $ make install
      $ npm link
