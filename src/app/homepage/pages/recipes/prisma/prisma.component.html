Prisma

Prisma is a performant open-source GraphQL layer doing the heavy lifting in your GraphQL server. In this chapter we won't go into details about Prisma, so head over to their website and have a look what features are available.

Hint In this article, you'll learn how to integrate Prisma into the Nest framework. We assume that you are already familiar with GraphQL concepts and the @nestjs/graphql module.

Setup Prisma

While working with Prisma you can either host your own instance or use the Prisma Cloud. In this introduction we are going to use the demo server provided by Prisma.

  1. Install the Prisma CLI: npm install -g prisma
  2. Create a new service prisma init and choose the demo server and follow the instructions.
  3. Deploy your service: prisma deploy

If you find yourself in trouble jump over to their quickstart section for furhter details. Eventually you should see two new files in your project directory.

{{graphqlconfigYml}}

To download the Prisma database schema to src/prisma/prisma-types.graphql and create a Prisma client under src/prisma/prisma.binding.graphql, run the following commands in your terminal:


$ graphql get-schema --project database
$ graphql codegen --project database
