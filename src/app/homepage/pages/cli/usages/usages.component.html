Usages

This section explain how to use NestjsCli.

Commands

    
      $ nest [command] [...options]

Options

Each command can accept the list of options below:

  • --dry-run : allow to simulate the command execution to see how it will affect your working directory.

new

The new command generates a standard Nestjs project structure as well as install the packages from npm package manager.

Option Name Description Required Default Value
directory The directory name to create the app in. true N/A

generate

The generate command generates a Nestjs architecture component.

Option Name Description Required Default Value
schematic The schematic name from the @nestjs/schematics collection. true N/A
name The name of the generated Nestjs architecture component. true N/A
path The path from to generate the Nestjs architecture component. false src
